What is the DOM?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

What is event bubbles?

The bubbles read-only property of the Event interface indicates whether the event bubbles up through the DOM tree or not.

The bubbles event property returns true if an event is a bubbling event.Otherwise it returns false.


What is callback function and what does it do?

In programming, a callback function is a function that is passed as an argument to another function and is called when the other function has completed its task.
